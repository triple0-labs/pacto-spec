# Pacto de Planes

Creado por: `pacto-cli`  
Idioma oficial para documentos Pacto: **español**

## Propósito

Definir una estructura formal y consistente para crear, revisar estado y ejecutar planes.

## Alcance del sistema

- Estados de plan: `current`, `to-implement`, `done`, `outdated`
- Índice global: `README.md`
- Plantilla oficial: `PLANTILLA_PACTO_PLAN.md`
- Comandos slash: `SLASH_COMMANDS.md`

## Reglas mínimas

1. Toda actualización de estado debe tener fecha absoluta (`YYYY-MM-DD`).
2. El estado declarado debe tener evidencia verificable.
3. Al crear/mover/eliminar planes, actualizar `README.md`.
